入れ子になった辞書構造体

```
> c "maps/nested.exs"
[BugReport, Customer]
> report = %BugReport{owner: %Customer{name: "Dave", company: "Pragmatic"}, details: "broken"}
%BugReport{
  details: "broken",
  owner: %Customer{company: "Pragmatic", name: "Dave"},
  severity: 1
}
```

ドット記法で入れ子の構造体にアクセスできる

```
> report.owner.company
"Pragmatic"
```
