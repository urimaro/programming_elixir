# 20.9 とんでもなく深掘り

内部表現(抽象構文木)は、3つの要素のタプル

1. 関数 or マクロ
2. メタデータ
3. 引数のリスト

```
> quote do: 1 + 2
{:+, [context: Elixir, import: Kernel], [1, 2]}
```

`Code.eval_quoted` で実行できる  
メタデータは省略できる

```
> Code.eval_quoted {:+, [], [1, 2]}
{3, []}
```

コードはただのタプル  
関数の定義を変えたり、実行中に新しいコードを作れる  
変数のスコープを制御できるので、安全に行える
