10.4 内包表記

```
> for x <- [1, 2, 3, 4, 5], do: x * x
[1, 4, 9, 16, 25]
> for x <- [1, 2, 3, 4, 5], x < 4, do: x * x
[1, 4, 9]
```

ジェネレータ(generator): コレクションから値を展開する方法を指定する

pattern <- enumerable_thing

ジェネレータが2つある場合、入れ子になる

```
> for x <- [1, 2], y <- [5, 6], do: x * y
[5, 6, 10, 12]
> for x <- [1, 2], y <- [5, 6], do: {x, y}
[{1, 5}, {1, 6}, {2, 5}, {2, 6}]
```
