**Listはheadとtailからできている** とも考えられる

```
> [3 | []]
[3]
> [2 | [3 | []]]
[2, 3]
```

```
> [1 | [2 | [3 | []]]]
[1, 2, 3]
> [a, b, c] = [1, 2, 3]
[1, 2, 3]
> a
1
> b
2
> c
3
> [head | tail] = [1, 2, 3]
[1, 2, 3]
> head
1
> tail
[2, 3]
```

文字列を表現する方法は2つある
- `"` でくくる: リテラル
- `'` でくくる: 整数のリスト  
    文字に変換できない要素がある => 整数のリスト

```
> [99, 97, 116]
'cat'
> [99, 97, 116, 0]
[99, 97, 116, 0]
```
