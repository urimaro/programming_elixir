# 真偽値

```
> :true == (1.0 == 1)
true
> :false == (1.0 === 1)
true
```

# 演算子
## 比較演算子

```
> 1.0 === 1
false
> 1.0 == 1
true
> 1.0 !== 1
true
> 1.0 != 1
false
```

`!!`で真偽値に変換

```
> a = 1
1
> !a
false
> !!a
true
> b = nil
nil
> !b
true
> !!b
false
> !!nil == false
true
> !!nil === false
true
```

## 算術演算子

```
> 5 / 2
2.5
> div 5, 2
2
> rem 5, 2
1
> rem -5, 2
-1
> rem 5, -2
1
```
